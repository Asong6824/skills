# Life Essay Writing Workflow

Generates life essays in the style of Jugaku Akiko, focusing on material details and craftsmanship.

## Skill Location

`~/.agent/skills/life-essay-writing/SKILL.md`

## Core Steps

1. **Analyze Request**: Identify the object or memory to be described.
2. **Material Focus**: Apply `rules/material-focus.md` to describe textures and origins.
3. **Craft Archeology**: Apply `rules/craft-archeology.md` to structure the narrative around the object's creation/maintenance.
4. **Tone Calibration**: Apply `rules/tactile-tone.md` to ensure precise, tactile language.
