# culture-song-quote

在文中穿插当地歌谣、标语或谚语。

## 为什么重要

地方性的声音比作者的解释更能传达文化氛围。

## ❌ 错误示范

```
这个地区有很悠久的渔业传统，当地人世代以捕鱼为生，对大海有着深厚的感情。
```

问题：说教式的文化介绍，没有生命力。

## ✅ 正确示范

```
浮冰融化，春風拂拂
薔薇綻放，海鷗嚶鳴
海洋遠處異國輪船的蒸汽
真教宗谷岬人們喜出望外
```

```
路邊標語寫著：「鄉間無醫，務請慎駕」。
```

## 技巧

- 直接引用，不要解释
- 包括街头标语、店铺招牌
- 引用后可以简短评论，但不要分析
- 方言和俗语最有力量
