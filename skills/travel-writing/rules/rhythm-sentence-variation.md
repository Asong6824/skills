# rhythm-sentence-variation

长句写景，短句写人物动作。交替使用创造节奏。

## 为什么重要

单一的句式让读者疲倦。变化的节奏创造呼吸感。

## ❌ 错误示范

```
中午我走在田间。阳光很强。一个女人在收割。一个男人向我招手。天很热。我继续走。
```

问题：全是短句，像机器人日志。

## ✅ 正确示范

```
晌午，開始轉往內陸田埂，在翠綠耀眼的稻田間走了十八公里。炙熱的陽光遍灑在碧藍色筒倉與荷蘭式穀倉屋頂上。一個女人戴著白色頭巾，操作牽引機。男人用耙叉把乾草耙成堆，向我招招手，嘴裡叼了根香菸。
```

## 模式

1. **长句** → 设置场景、氛围、时间
2. **中句** → 视觉细节、环境描写
3. **短句** → 人物动作、情绪节点

## 技巧

- 长句用于铺陈
- 短句用于动作和转折
- 重要时刻用最短的句子
