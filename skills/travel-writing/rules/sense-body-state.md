# sense-body-state

用身体感受锚定叙事：疲劳、饥渴、疼痛、温度。

## 为什么重要

身体状态让读者感同身受，体验旅途的真实。

## ❌ 错误示范

```
今天走了很久，有点累。
```

问题：太笼统，读者无法感受。

## ✅ 正确示范

```
頭兩天，我走了六十多公里，一路在毫無蔽蔭的曠野中，頂著空前的熱浪前行。路上唯一的陰影往往是在人們漠然臉孔的凹凸處。
```

```
熱水輕撫著我的雙腿、肩膀和胃部。
```

```
雙腳起了水泡，每一步都像踩在碎玻璃上。
```

## 身体状态词汇表

| 类型 | 词汇 |
|-----|-----|
| 疲惫 | 沉重、发软、酸痛、麻木 |
| 饥饿 | 空洞、绞痛、咕咕叫 |
| 热 | 灼烧、蒸烤、汗水刺眼 |
| 冷 | 刺骨、僵硬、发抖 |
| 疼痛 | 刺痛、撕裂、钝痛 |

## 技巧

- 具体部位 > 整体感受（双腿酸痛 > 累）
- 用比喻描述疼痛
- 身体感受可以成为段落的锚点
