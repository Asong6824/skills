# human-action-portrait

通过一个动作刻画人物，而非形容词堆砌。

## 为什么重要

动作比形容词更可信，让读者自己得出结论。

## ❌ 错误示范

```
老板是个细心、热情、善良的人。
```

问题：空洞的形容词，读者无法"看到"这个人。

## ✅ 正确示范

```
老闆小心翼翼地打開手帕，在右上角用印章蓋上民宿的地址與電話號碼……又在日期下面用紅筆寫了一行小字：「七點出發」。
```

```
一位老婆婆綻放出非常可愛的微笑，整張臉神采飛揚……她很瘦小，有個難看的駝背，就坐在一台別人送給她的大型電子編織機前面。
```

## 技巧

- 选择一个最能代表人物的动作
- 加入一个独特的外貌细节
- 让动作自己说话，不要解释含义
