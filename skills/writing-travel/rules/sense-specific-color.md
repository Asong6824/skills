# sense-specific-color

使用精确颜色，避免泛泛的"蓝"、"绿"、"红"。

## 为什么重要

泛泛的颜色词无法在读者脑中形成画面。精确的颜色创造感官记忆。

## ❌ 错误示范

```
海面很蓝，天空很蓝。
山上的树很绿。
```

问题：读者无法区分海蓝和天蓝，"绿"没有质感。

## ✅ 正确示范

```
海面不見絲毫波濤或漣漪，一片亮藍刺眼，像柳橙廣告中的海洋。
```

```
翠綠耀眼的稻田間，炙熱的陽光遍灑在碧藍色筒倉與荷蘭式穀倉屋頂上。
```

## 技巧

- 用形容词修饰颜色：亮蓝、翠绿、暗红、粉白
- 用比喻锚定颜色：像柳橙广告、像旧报纸
- 注意光线对颜色的影响：阳光下的蓝 vs 阴影下的蓝
