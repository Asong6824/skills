# human-gift-record

记录旅途中收到的小恩惠，不煽情。

## 为什么重要

小礼物和善意是旅行中最珍贵的记忆，但过度抒情会破坏真实感。

## ❌ 错误示范

```
他递给我一瓶水，我感动得热泪盈眶，这份陌生人的善意让我对人性重拾信心……
```

问题：过度解读，煽情。

## ✅ 正确示范

```
一輛牽引機經過，年輕司機停下車，一語不發地遞給我一瓶柳橙汁。我站在馬路中央將它一口飲盡。
```

```
她低頭看看我的靴子，嘆了一口氣，將黃瓜放到我手裡。「你的腳一定很痛吧。」她說。黃瓜冰涼，帶著一股淡淡的米糠香。
```

## 技巧

- 记录给予的动作，不解释动机
- 用身体反应代替情感抒发（一口饮尽）
- 加入礼物的感官细节（冰凉、香味）
- 让读者自己感动
