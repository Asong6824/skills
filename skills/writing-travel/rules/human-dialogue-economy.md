# human-dialogue-economy

保持对话简短，不加修饰语和解释。

## 为什么重要

真实对话没有"愤怒地说"、"好奇地问"。简洁的对话更有力量。

## ❌ 错误示范

```
"你要走多远？"他好奇地问道，眼睛里闪着惊讶的光芒。
"整个日本那么远。"我自豪地回答，心里充满了期待。
```

问题：修饰语削弱对话本身的力量。

## ✅ 正确示范

```
「你明早要幾點起床？」
「我不知道，七點吧。」
「要趕巴士嗎？」
「不是，我要徒步。」
「要走很遠嗎？」
「整個日本那麼遠。」
```

## 技巧

- 删除所有"地说"前面的副词
- 删除对话后的心理描写
- 让对话内容本身传达情绪
- 捕捉方言或口语特色
