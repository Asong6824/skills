# 游记写作风格指南

本目录下的所有写作任务默认使用「亚兰·布斯」风格。

```

```

## 快速参考

### 优先级

1. **感官锚定** (CRITICAL) - `sense-*`
2. **人物连接** (CRITICAL) - `human-*`
3. **叙事节奏** (HIGH) - `rhythm-*`
4. **文化融入** (MEDIUM) - `culture-*`

### 核心原则

- 用精确的感官细节代替抽象形容词
- 通过动作（非形容词）刻画人物
- 对话简短，不加修饰语
- 记录身体感受和小恩惠
- 穿插当地歌谣、标语
- 保持温暖基调，避免讽刺
